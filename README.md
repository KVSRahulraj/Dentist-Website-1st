# Veda Dental Studio - Ultra-Premium Landing Page

A high-fidelity, high-performance React landing page for a modern dental clinic. Designed with a focus on trust, technology, and patient care.

##  Features

- **Premium Aesthetic**: Responsive design using Deep Navy, Royal Blue, and Slate White tones.
- **Fast-Track Booking**: Glassmorphic booking form with high-contrast inputs for better accessibility.
- **Interactive Testimonials**: Continuous sideways-sliding marquee that pauses on hover to showcase patient trust.
- **SEO Optimized FAQs**: 15 comprehensive questions addressing patient concerns and clinical services with auto-close logic.
- **Modern Tech Stack**: Built with React 19, Vite, Tailwind CSS, and Lucide React icons.

##  Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm (comes with Node.js)

### Installation

1. Open a terminal in the project directory:
   ```bash
   cd veda-dental-studio---ultra-premium-landing-page
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser to `http://localhost:3000` (or the port specified in your terminal).

##  Project Structure

- `components/`: Modular UI sections (Hero, Services, Testimonials, FAQ, etc.).
- `App.tsx`: Main layout and component integration.
- `index.html`: Entry point with custom Google Fonts and global CSS resets.
- `vite.config.ts`: Vite configuration for performance and path aliasing.

##  Deployment

This project is optimized for deployment on [Vercel](https://vercel.com) or [Netlify](https://netlify.com).

1. Push your code to a GitHub repository.
2. Connect the repository to your hosting provider.
3. The build command is `npm run build` and the output directory is `dist`.
